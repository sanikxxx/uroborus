# Digital Ruble Agent — MVP

Минимально жизнеспособный агент для работы с цифровым рублем, построенный на принципах скилов и RAG-поиска.

## Архитектура MVP

Агент имеет две функции:

1. **Чтение инструкции** — загрузка правил и инструкций из базы знаний
2. **Выполнение функций** — обработка запросов с использованием registered skills и RAG

### Фазы развития

| Фаза | Описание |
|------|----------|
| **Phase 1: Documentation** | База знаний и RAG-поиск |
| **Phase 2: Demonstrations** | Демонстрации возможностей |
| **Phase 3: Tools** | Инструменты и API |

## Стек

- **Python 3.11+**
- **OLLAMA** — локальная LLM модель (Qwen3-Coder-Next по умолчанию)
- **sentence-transformers** — эмбеддинги для RAG
- **PyYAML** — загрузка знаний

## Принципы работы

1. **Skills-first**: каждый ответ — результат выполнения зарегистрированного скилла
2. **RAG-выборка**: при запросе агент выбирает релевантные документы, а не передаёт весь контекст
3. **Инкрементальная разработка**: каждый скилл — отдельный модуль, легко добавлять и тестировать
4. **Логирование**: все запросы и ответы сохраняются для последующего анализа

## MVP-фичи

- RAG-поиск по базе знаний (русский язык)
- Базовая система скилов (простой dispatch по ключевым словам)
- API-логирование (GET endpoint для сбора логов)
